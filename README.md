      
 # Students Directory - React + Tailwind

A modern, responsive React application that displays a list of users fetched from an API and shows detailed information for each user. The UI is built with **Tailwind CSS** and includes smooth hover effects, shadows, and consistent font styling.

## Features

- Fetches user data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users)
- Responsive grid layout for user cards
- Hover effects and shadow animations for cards
- Clicking a card navigates to a **full details page** using React Router
- Consistent fonts and styling across cards and details page
- Back button to return from the details page
